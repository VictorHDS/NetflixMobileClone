# Maratona Clonando a Netflix

## Technologies Used:
* HTML;
* CSS;
* GIT;
* GitHub;
* Vercel (Deploy).

## Standardizations:
* HTML: Semantic Tags;
* CSS: BEM (Block Element Modifier);
* GIT: Conventional Commits.
